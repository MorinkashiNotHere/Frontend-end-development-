### `Headings`

allows you to display titles and subtitles in web pages

+ Syntax : 

        <body>
            <h1>Heading 1</h1>
            <h2>Heading 2</h2>
            <h3>Heading 3</h3>
            <h4>Heading 4</h4>
            <h5>Heading 5</h5>
            <h6>Heading 6</h6>
        </body>

+ Ex : 

><body>
><h1>Heading 1</h1>
><h2>Heading 2</h2>
><h3>Heading 3</h3>
><h4>Heading 4</h4>
><h5>Heading 5</h5>
><h6>Heading 6</h6>
></body>

### `paragraphs`
are used for displaying text content on a webpage. It is defined by the `<p>`

+ Syntax :

        <p>
            This paragraph
            contains a lot of lines
            but they are ignored.
        </p>

+ EX : 
  <p>
            This paragraph
            contains a lot of lines
            but they are ignored and you can go an check the code 
        </p>

>**Note** : that putting content on a new line is ignored by the web browser.
